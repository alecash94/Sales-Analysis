# Sales-Analysis
Analisi dei dati di vendita con Python ed Excel. Il progetto include pulizia dei dati, analisi esplorativa (EDA) e creazione di una dashboard interattiva per analizzare trend temporali e performance per categoria e sottocategoria.
# 📊 Analisi Vendite con Python ed Excel

## 📌 Descrizione del progetto
Questo progetto analizza un dataset di vendite con l’obiettivo di individuare trend temporali e comprendere quali categorie e sottocategorie contribuiscono maggiormente al fatturato.

L’analisi è stata realizzata utilizzando Python per l’elaborazione dei dati ed Excel per la creazione di una dashboard interattiva.

---

## 🛠 Strumenti utilizzati
- Python (Pandas, Matplotlib, Seaborn)
- Excel (Tabelle Pivot, Grafici)
- Dataset Kaggle

---

## 📊 Attività svolte

### 🔹 Pulizia e preparazione dati
- Conversione e gestione delle date
- Creazione della variabile “anno ordine”
- Rimozione e gestione dei dati mancanti

### 🔹 Analisi esplorativa (EDA)
- Analisi delle vendite nel tempo
- Analisi per categoria e sottocategoria
- Identificazione dei principali driver di vendita

### 🔹 Visualizzazione dati
- Grafici a linee per l’analisi dei trend temporali
- Grafici a barre per il confronto tra sottocategorie
- Dashboard interattiva in Excel tramite pivot table

---

## 📈 Insight principali
- Le vendite mostrano una crescita costante nel periodo analizzato
- Alcune categorie contribuiscono in modo predominante al fatturato
- Poche sottocategorie generano la maggior parte delle vendite
- L’analisi per livello gerarchico (categoria → sottocategoria) permette una lettura più approfondita dei dati
